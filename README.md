# Imperium

帝國是一個基於 PHP 的權限管理系統，建立一個帝國，需要細心的規劃，

與其他權限管理系統不一樣的地方在於：**帝國是基於組織，與資源條件的系統。**

如果你不清楚，我們稍候會提到什麼是資源條件。

&nbsp;

# 特色

1. 動作請求紀錄

2. 明瞭的使用方式

3. 動作捷徑

4. 角色系統

5. 組織系統

6. 單一個使用者可擁有多重角色

7. 資源條件設置
